# Chirag Farmer

Chirag Farmer is an Android app built to make everyday farming tasks a little easier. It brings together different services that farmers often need, such as buying and selling agricultural products, booking drone spraying services, and detecting crop diseases using AI.

I built this project because I grew up seeing the challenges farmers face every day. Finding quality products, reaching the right buyers, and getting timely advice when crops become unhealthy isn't always easy. I wanted to create something that could simplify these tasks and make useful technology more accessible.

## Features

* Buy and sell agricultural products through an easy-to-use marketplace
* Book drone spraying services directly from the app
* Detect crop diseases using AI by uploading a photo of the affected plant
* Secure online payments with Razorpay
* Location support using OpenStreetMap
* Cloud-based image storage with Cloudinary
* Clean and simple interface designed for everyday use

## Tech Stack

* Kotlin
* MVVM Architecture
* Clean Architecture
* Firebase
* Razorpay
* Cloudinary
* OpenStreetMap

## About the Project

This project started as a simple marketplace where farmers could buy and sell agricultural products. As I continued working on it, I realized there were many other problems that could be solved within the same app. That's when I added features like drone service booking and AI-based crop disease detection.

As the project grew, I migrated the codebase to Clean Architecture with MVVM to keep it organized, scalable, and easier to maintain. Building Chirag Farmer taught me a lot about Android development, app architecture, API integration, payment gateways, and designing applications around real-world problems.

I'm still improving the project by adding new features and refining the overall experience.
